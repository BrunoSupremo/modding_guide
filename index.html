Stonehearth's Modding Guide

Welcome to the Stonehearth modding guide!

This guide is still work in progress, so there are many missing pages and not all the links work yet.

We've divided the guide in several groups:

  • The modding guide : a "how-to" approach to modding, step by step.
  • Scripting API reference : code documentation, including the C++ components.

The scripting API reference is not ready yet, and depending on the length of other sections of the guide we might add a third group here (a "Programmer's guide") explaining the interface of different types of Lua scripts that we can find in the game.

We'll be updating the modding guide in the future to add all the missing content and fixing styling / formatting issues. Right now, only the essentials are finished. Make sure to read them before attempting advanced stuff.

Happy modding!
